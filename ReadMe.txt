Setup apache + php + mysql server [ex. xampp]
Put all the files in htdocs folder

1. To setup mysql db, import the provided sql file in mysql console.

2. for offline tweets fetching : 
from browser: http://localhost/TweetFetcher.php?hashtag=India
or you can run it through curl [curl "url"]

3. For online view of recent tweets (auto- refreshed)
User can go to http://localhost/index.html and provide desired hashtag.
